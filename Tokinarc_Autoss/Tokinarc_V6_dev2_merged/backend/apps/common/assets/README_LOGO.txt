=== CHỖ ĐẶT LOGO CÔNG TY CHO CHỨNG TỪ XUẤT (Excel / Word) ===

Bỏ file logo của AUTOSS vào đúng thư mục này, ĐẶT TÊN CHÍNH XÁC:

    autoss_logo.png

(hoặc autoss_logo.jpg — code sẽ tự tìm .png trước, rồi .jpg)

Cách bỏ file vào (không cần gửi qua chat):
  1) Mở Windows Explorer tới thư mục:
     ...\backend\apps\common\assets\
     rồi copy/paste file logo vào, đổi tên thành autoss_logo.png
  HOẶC
  2) Kéo-thả file logo từ Explorer vào panel EXPLORER của VS Code,
     thả vào thư mục apps/common/assets, rồi đổi tên thành autoss_logo.png

Khuyến nghị: ảnh nền trong suốt (PNG), chiều ngang ~300–600px, tỷ lệ ngang.

Nếu KHÔNG có file này, chứng từ vẫn xuất bình thường — chỉ dùng
tiêu đề chữ "CÔNG TY TNHH THƯƠNG MẠI DỊCH VỤ AUTOSS" thay cho logo.
