# Scripts

Utility scripts — chạy thủ công, không phải server code.

- `add_product.py` — Thêm part mới vào data JSON
- `apply_catalog_aliases.py` — Batch update P/D aliases từ catalog PDF
- `catalog_aliases.py` — Mapping data từ CATALOG_02.pdf
- `query_engine.py` — CLI query tool
- `show_fails.py` — Hiện failed eval cases
- `check_system.py` — Audit file dependencies
